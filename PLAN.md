# План разработки MVP магазина приложений для Android

**Проект:** MVP магазин приложений  
**Платформа:** Android (Kotlin, minSdk 24)  
**Архитектура:** MVVM, Clean Architecture  
**Технологии:** Coroutines, Room, Retrofit, Hilt, Jetpack Compose, Coil
